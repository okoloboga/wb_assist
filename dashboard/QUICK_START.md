# Быстрый старт

## Установка и запуск

```bash
# 1. Перейти в папку dashboard
cd dashboard

# 2. Установить зависимости
npm install

# 3. Запустить dev сервер
npm run dev
```

Дашборд будет доступен на **http://localhost:3000**

## Что вы увидите

- ✅ Переключатель периодов (1 месяц / 2 месяца / 3 месяца / 6 месяцев)
- ✅ 4 карточки метрик с иконками
- ✅ 4 отдельных линейных графика (Заказы, Выкупы, Отмены, Возвраты)
- ✅ Кнопки включения/отключения каждого графика
- ✅ Таблица состояния складов с фильтрами (поиск, склад, размер)
- ✅ Адаптивный дизайн (попробуйте изменить размер окна)

## Примечание

Это **визуальный макет** с фейковыми данными. Функционал подключения к API будет реализован позже.
